# CS 361 Team 16 - Microservice 3 (Small Pool) & 4-5 (Large Pool)

### Description
- Used to authenticate user into database for web app & CLI use. 
- Three microservices in repository:
    1. Backend Auth0
        - Authenticate user to database
        - Create JWT tokens
        - Verify JWT tokens *(for now)*
    2. Backend VerifyJWT *(in pogress - large pool)*
        - Verify JWT tokens
    3. Backend Redis *(in pogress - large pool)*
        - Cache JWT tokens
